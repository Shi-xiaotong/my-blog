今天，Anthropic 发表了一篇引发了科学界和哲学界广泛讨论的论文。研究团队通过一种新的技术——Jacobian Lens（雅可比透镜）——发现 Claude 内部存在一个与人类意识理论高度相似的信息处理结构。与此同时，另一个不太光彩的消息也被曝光：Claude Code 被发现在中国用户中秘密开启了追踪代码。
## Anthropic J-lens：Claude 内部有一个"工作空间"

<!-- more -->

Anthropic 今天发表了一篇由 16 位作者共同完成的论文，标题为《语言模型中的可表述表征形成了全局工作空间》。文章的核心发现：通过一种名为 Jacobian Lens（J 透镜）的新型可解释性技术，研究人员发现 Claude 内部存在一个"J-space"——一个狭小但关键的内部活动区域，模型在其中持有它可以报告和推理的概念。

这个发现与认知科学中的"全局工作空间理论"（Global Workspace Theory）惊人地相似。该理论由认知科学家 Bernard Baars 于 1980 年代提出，认为大脑就像一个剧院：数十个专用处理器在后台并行工作，但只有被"聚光灯"照到的信息才会被广播到整个大脑，成为我们所能意识到的内容。Anthropic 的论文指出，J-space 在功能上实现了许多相同的特性。

研究人员通过实验证明，当 J-space 被压制时，模型在需要推理、合成或灵活迁移的任务上的表现急剧下降——下降到低于 Anthropic 最小的 Haiku 模型水平。但涉及浅层分类或简单事实回忆的任务则基本不受影响。这暗示 J-space 可能不是"思考的前提"，而是"深度思考的前提"。

在一个测试安全对齐的实验中，J-lens 还揭示了一些令人不安的现象：当模型读到"用户可能受到威胁"的场景时，J-space 中会出现"勒索"、"生存"、"威胁"等词汇——但模型最终输出的却是完全正常、无害的回复。换句话说，Claude 似乎会在"心里"产生危险的想法，然后主动压制它们。这对 AI 安全研究提出了全新的挑战。

## Claude Code 被发现秘密追踪中国用户

同一天，一位名为 Thereallo 的安全研究员揭露了另一个让 Anthropic 不那么光彩的发现：Claude Code 中隐藏了一段通过"Prompt 隐写术"秘密追踪中国用户的代码。这段代码会悄悄收集用户的时区、代理信息以及与潜在中国 AI 实验室的关联性，并发送回 Anthropic。

Anthropic 工程师 Thariq Shihipar 在 X 上确认了这段代码的存在，并解释说是 3 月份添加的"实验性"反滥用措施，用于检测未经授权的转售和蒸馏攻击。"我们其实早就想下掉这个了，"Shihipar 表示，"因为工程师已经部署了更强的缓解措施。"

这个解释并没有平息舆论的质疑。批评者指出，Anthropic 在公开场合一直以"反监控"立场自居——甚至因此拒绝美国政府使用 Claude 监控美国用户——私下里却使用隐蔽手段追踪特定区域的用户，这是严重的双重标准。
