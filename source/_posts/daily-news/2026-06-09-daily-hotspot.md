Cohere 今天一次性发布了两款新模型——North Mini Code 和 Command A+。在多伦多的这家公司一向低调，但每次出手都很准。这次的产品组合很有意思：一个专攻编码和 agent 任务，一个覆盖多语言多模态推理。显然，Cohere 在走一条不同于 OpenAI 和 Anthropic 的路。
## Cohere 发布 North Mini Code：专为 agent 场景打造的 30B 编码模型

<!-- more -->

North Mini Code 是一个 30B 参数的 MoE（混合专家）模型，但激活的参数只有 3B。这个设计让它非常高效——对于编码和 agent 编程任务，Cohere 宣称其性能接近 7B 级别的密集型模型，但推理成本只有后者的不到一半。

为什么这个定位有趣？因为"编码 agent"是 2026 年 AI 应用层最热门的场景之一。Claude Code、GitHub Copilot、Cursor 等产品都在抢这个市场，但它们背后运行的都是大几百亿甚至上千亿参数的大模型。Cohere 的这个模型暗示了一个趋势：专用的小参数模型在特定任务上完全可以匹敌通用大模型，而且成本优势巨大。

## Command A+：218B MoE 的多语言多模态全能手

与 North Mini Code 同时发布的是 Command A+，一款 218B 参数的 MoE 模型（25B 激活）。这个模型的最大卖点是支持 48 种语言和视觉推理能力。Cohere 一直以来在企业多语言场景上都有积累，这次直接把这个优势放大到了视觉领域。

Command A+ 还特别强化了工具使用和 agent 能力。Cohere 的 CTO 在发布会上展示了一个 demo：用自然语言让模型调用多个 API 完成一个复杂的跨国数据分析任务，整个过程不需要人工干预。这对于那些需要在多语言环境中部署 AI agent 的企业来说，是一个很有吸引力的选择。
