今天的科技圈可以说是被 Computex 霸屏了。NVIDIA 在台北电脑展上扔出了一堆重磅新品，从桌面级的 AI 芯片到数据中心的超算平台，覆盖了你能想到的每一个角落。与此同时，Alphabet 宣布了科技史上最大规模的股权融资，GitHub 则在开发者社区引爆了一颗按量计费的炸弹。三件事放在一起看，线索很清晰——整个行业正在为 AI 时代的基础设施疯狂砸钱。
## NVIDIA 联手微软重新定义 AI PC

<!-- more -->

NVIDIA 在 Computex 上正式发布了 RTX Spark 超级芯片，这是一颗将 Blackwell GPU 与 Grace CPU 封装在一起的怪物级 SoC。它的规格相当夸张：1 petaflop 的 AI 算力（是的，你没看错，一台笔记本大小的设备就能提供千万亿次级别的 AI 计算）、128GB 的统一内存、以及为 PC 形态优化的功耗控制。

更值得关注的是 NVIDIA 与微软的深度合作。双方联合宣布，将基于 RTX Spark 重新定义 Windows PC 的 AI 能力。微软正在将 Copilot Runtime 深度集成到 Windows 内核中，而 RTX Spark 将为这些端侧 AI 功能提供硬件加速。想象一下：你可以在笔记本上本地运行 70B 参数的大模型，不需要联网，不需要云端推理，数据也不离开你的设备。

首批 RTX Spark 整机预计在 2026 年秋季上市。惠普、戴尔、联想等 OEM 厂商均已宣布相关产品计划。价格方面，NVIDIA 尚未公布具体数字，但考虑到其定位——介于普通笔记本和专业工作站之间——预计售价将在 2000 到 4000 美元区间。

## Alphabet 宣布 800 亿美元 AI 基建融资

同一天，Google 母公司 Alphabet 宣布了一项规模惊人的股权融资计划：800 亿美元。你没看错，800 亿。这笔融资分为三块：300 亿美元的公开发行（含普通股和强制可转换优先股）、一个 400 亿美元的 ATM（按市价发行）计划、以及 100 亿美元面向 Berkshire Hathaway 的私募配售。

根据 Alphabet 提交的监管文件，这些资金将绝大部分用于 AI 基础设施的扩张，包括新建数据中心、采购 GPU 和自研芯片（TPU）、以及全球光纤网络的建设。CFO 在投资者电话会上透露，2026 年的资本支出指引为 1800 到 1900 亿美元，而 2027 年将进一步大幅增加。

这其中最引人注目的莫过于 Berkshire Hathaway 的参与。巴菲特一向以规避科技泡沫闻名，这次的 100 亿美元押注释放了一个强烈信号：AI 基础设施不是昙花一现的投机，而是像铁路、电网一样值得长期持有的资产。

## GitHub Copilot 改为按量计费引发开发者地震

对于广大开发者来说，今天最切身相关的消息可能是这个：GitHub Copilot 正式告别了固定订阅制，改为按量计费的 Credit 系统。新的计费规则是：1 AI Credit = $0.01，按模型消耗的 token 数收费。代码补全和 Next Edit 建议对付费用户仍然免费，但 agent 会话、前置模型访问、多步自主任务、代码审查等功能统统进入了计费范围。

结果很快在社区引发了海啸。Reddit、X 和 GitHub 论坛上贴出了大量账单截图：有用户从原来的 $29/月涨到了 $750，重度用户更是突破了 $3000。一条仅写着"Goodbye, Copilot"的帖子被分享了一万多次。一位用户算了一笔账——$3000 一个月，够买三台 MacBook Pro 了。

GitHub 首席产品官 Mario Rodriguez 的态度相当强硬："Copilot 已经不是一年前的那个产品了。"他的意思很明确——随着模型能力的提升和使用场景的扩展，按固定月费无限使用的模式是不可持续的。但开发者们显然不买账，Cursor、Claude Code 等竞品在消息公布后的注册量出现了明显增长。
